## How the project was built

Built upon [Anima App](https://www.animaapp.com) exported code from Zhao, Billie's Figma design.

Code export option: Vue.js with SASS.

## Develop

```
npm install
npm run start
```

## Deploy

Build `/dist` first:

```
npm run build
```

Then deploy `/dist` to your web server.
